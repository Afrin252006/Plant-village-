🌱 Plant Disease Detection using Deep Learning

📌 Overview

This project uses Convolutional Neural Networks (CNN) to detect plant diseases from leaf images.

📊 Dataset

- Dataset: PlantVillage Dataset (Kaggle)
- 54,000+ images
- 38 classes

🧠 Model

- CNN / MobileNetV2 (Transfer Learning)
- Accuracy: ~XX%

⚙️ Tech Stack

- Python
- TensorFlow / Keras
- OpenCV

🚀 How to Run

pip install -r requirements.txt
python train.py

🌾 Applications

- Smart farming
- Early disease detection
- Helps farmers increase yield
